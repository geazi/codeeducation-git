Meu primeiro repositorio no gitHub
==================================

Passo-a-passo após o commit
---------------------------

1. Criado o repositório no gitHub
2. Primeira tentativa de configurar o repositório remoto em nosso desktop, com o comando git remote add origin https://github.com/geazi/codeeducation-git.git
3. Na primeira tentativa obtivemos um erro em decorrencia de não estar posicionado dentro do diretório do repositório local.
4. Posicionado na pasta correta, dentro do repositório local.
5. Nova tentativa e sucesso: git remote add origin https://github.com/geazi/codeeducation-git.git
6. Tentativa de push no repositório remoto: git push origin master
7. Checado no gitHub se os arquivos tinham sido adicionados no repositório remoto. Tudo ok.
